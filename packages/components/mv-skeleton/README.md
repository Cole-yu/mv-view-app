## mv-skeleton 组件

#### props
| 属性 | 说明 | 类型 | 默认值 |
| :------: | :----: | :----: | :--: |
| height | 默认骨架屏高度 | String | 'auto' |
| direction | 默认骨架屏的骨架（灰色块）方向 | String | 'row' |
| rows | 默认骨架屏水平方向时骨架数量 | Number | 3 |
| showTitle | 默认骨架屏是否显示标题骨架 | Boolean | true |
| columns | 默认骨架屏垂直方向时骨架数量 | Number | 7 |
| loading | 是否显示骨架屏 | Boolean | false |
