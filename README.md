# mv-view-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Components
1. mv-scroll-list
```
左侧固定，表头固定的动态加载长列表组件
```
2. mv-ellipsis
```
文本超出省略组件
可控制行数，展开/收起，类似与评论查看
```
3. mv-skeleton
```
骨架屏组件
```
4. mv-datetime-picker
```
日历组件
2026-03-05
```
5. mv-dialog
```
命令式弹窗组件
```
6. mv-keyboard
```
命令式数字键盘组件
```
7. mv-reactive-txt
```
响应式文本组件
根据容器宽度自适应调整文本字体大小，不溢出容器范围，控制对齐方式
```
8. mv-scroll-bounce
```
滚动卡行高位置回弹列表组件
滚动动第一行和第二行之间，会自动回到到第一行（未超过一半）或第二行（超过一半）的起始位置
```
